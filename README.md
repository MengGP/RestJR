Используемые в проекте средства:
--------------------------------------
* Java SDK 1.8
* IntelleJ IDEA Community Edition 2020.2
* Система сборки и управления  зависимостями: Gradle 6.5
* Фреймворк: Spring Boot 2.2.4
* Веб-сервер: Apache Tomcat (средствами Spring Boot Web)
